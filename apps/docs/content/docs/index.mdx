# Commerce SDK

The Commerce SDK is a powerful toolkit for building Solana-powered commerce applications. It provides everything you need to integrate Solana Pay and blockchain functionality into your applications, from low-level utilities to high-level React components with full e-commerce features.

## Architecture

The Commerce SDK is built as a layered architecture with three main packages:

## Quick Start

### Comprehensive E-commerce Integration

```bash
pnpm add @solana-commerce/react-sdk
```

```tsx
import { SolanaCommerceSDK } from '@solana-commerce/react-sdk';

function MyStore() {
  return (
    <SolanaCommerceSDK
      config={{
        mode: 'cart',
        position: 'overlay',
        merchant: {
          name: 'My Store',
          wallet: 'YOUR_SOLANA_ADDRESS',
          description: 'Premium digital products'
        },
        theme: {
          primaryColor: '#9945FF',
          borderRadius: 'lg'
        },
        products: [
          {
            id: 'item-1',
            name: 'Premium Course',
            description: 'Complete blockchain development guide',
            price: 100000000, // 0.1 SOL
            currency: 'SOL'
          }
        ],
        allowedMints: ['SOL', 'USDC'],
        showQR: true,
        showProductDetails: true
      }}
      onPayment={(amount, currency, products) => {
        console.log('Payment initiated:', { amount, currency, products });
      }}
      onPaymentSuccess={(signature) => {
        console.log('Payment successful!', signature);
      }}
    />
  );
}
```

### Simple Tip Button

```tsx
import { SolanaPayButton } from '@solana-commerce/react-sdk';

function TipWidget() {
  return (
    <SolanaPayButton 
      recipient="YOUR_SOLANA_ADDRESS"
    />
  );
}
```

### Vanilla JavaScript

```bash
pnpm add @solana-commerce/headless-sdk
```

```javascript
import { 
  createCommercePaymentRequest,
  createCartPaymentRequest,
  STABLECOINS 
} from '@solana-commerce/headless-sdk';

// Create a cart payment request
const cartRequest = createCartPaymentRequest(
  'YOUR_WALLET_ADDRESS',
  [
    {
      id: 'product-1',
      name: 'Digital Art',
      price: 50000000, // 0.05 SOL
      currency: 'SOL'
    }
  ]
);

console.log('Payment URL:', cartRequest.url);
console.log('Total:', cartRequest.getAmountDisplay());
```
